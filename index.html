• h1{} - выбирает тип элементов, с значение определенного tag.
  • #title{} - выбирает элементы, основываясь на значении их id атрибута.
  • .title{} - выбирает элементы, основываясь на значении их атрибута class.
  • h1#title{} - выбирает элементы, основываясь на нескольких индификаторах. h1.title{} h1.title#title{} .title.test{}
  • .title .list{} - выбирает элемент внутри которого лежит другой элемент на любом уровне вложенности.
  • *{} - Выбирает все элементы.
  • div > p{} - выбирает элемент внутри которого непосредственно лежит другой элемент.
  • h1 + p{} - выбирает элемент рядом с которым лежит другой элемент.
  • h1 ~ p{} - выбирает все элементы после выбранного элемента.
  • [attr]{} - выбирает все элементы, имеющие данный атрибут или атрибут с определённым значением.
